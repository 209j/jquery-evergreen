toArray

Convert NodeList to Array.

Parameters:

  • collection can be a NodeList or an Array.

Returns an Array

var toArray = function(collection) { return [].slice.call(collection); };

makeIterable

Make sure to return something that can be iterated over (e.g. using forEach). Arrays and NodeLists are returned as-is, but Nodes are wrapped in a [].

Parameters:

  • element can be a Node, a NodeList, or an Array.

Returns an Array or a NodeList

var makeIterable = function(element) { return 'length' in element && element !== window ? element : [element]; }; export { global, toArray, makeIterable };
